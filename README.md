## 📌 Project Overview
This project aims to predict whether a person is at risk of heart disease using machine learning techniques. The model is trained on medical attributes such as age, cholesterol levels, heart rate, and more to classify patients into two categories:
- 0 → No Heart Disease  
- 1 → Heart Disease  

---

## 🎯 Objective
To build a reliable classification model that can assist in early detection of heart disease based on patient health data.

---

## 📊 Dataset
- **Source:** UCI Heart Disease Dataset (Kaggle)
- Contains medical features like:
  - Age
  - Sex
  - Chest Pain Type (cp)
  - Resting Blood Pressure (trestbps)
  - Cholesterol (chol)
  - Maximum Heart Rate (thalach)
  - And more...

---

## ⚙️ Technologies Used
- Python 🐍
- Pandas & NumPy (Data Handling)
- Matplotlib & Seaborn (Visualization)
- Scikit-learn (Machine Learning)

---

## 🔄 Project Workflow (Step-by-Step)

### 1️⃣ Data Loading
- Imported dataset using Pandas
- Checked structure using `.head()`, `.info()`, `.describe()`

---

### 2️⃣ Data Cleaning
- Identified missing values
- Removed or handled null values
- Converted target variable into binary classification (0 & 1)

---

### 3️⃣ Exploratory Data Analysis (EDA)
- Visualized data using:
  - Count plots (target distribution)
  - Heatmap (feature correlation)
- Identified relationships between features

---

### 4️⃣ Feature Selection
- Selected relevant features
- Removed unnecessary or non-numeric columns

---

### 5️⃣ Data Splitting
- Divided dataset into:
  - Training set (80%)
  - Testing set (20%)

---

### 6️⃣ Feature Scaling
- Applied StandardScaler to normalize data
- Improved model performance and convergence

---

### 7️⃣ Model Building
- Used **Logistic Regression** for classification
- Trained model on training dataset

---

### 8️⃣ Model Evaluation
Evaluated performance using:
- ✅ Accuracy Score  
- 📊 Confusion Matrix  
- 📈 ROC Curve & AUC Score  
- 📋 Classification Report  

---

### 9️⃣ Feature Importance Analysis
- Identified which features contribute most to prediction
- Visualized importance using bar chart

---

## 📈 Results & Insights
- The model achieved strong predictive performance
- ROC Curve indicates good classification ability
- Key influencing features include:
  - Chest pain type
  - Maximum heart rate
  - ST depression (oldpeak)

---

## 🚀 Conclusion
This project demonstrates how machine learning can be applied in healthcare to assist in early diagnosis of heart disease. With proper data preprocessing and model tuning, reliable predictions can be achieved.
